# Tech Stack Logos

Place technology/framework logos here.

## Naming Convention

Use lowercase tech names:

- `swift.png` or `swift.svg`
- `react.png` or `react.svg`
- `firebase.png` or `firebase.svg`
- `nodejs.png` or `nodejs.svg`
- `tailwind.png` or `tailwind.svg`
- `nextjs.png` or `nextjs.svg`
- `flutter.png` or `flutter.svg`
- `supabase.png` or `supabase.svg`
- `vercel.png` or `vercel.svg`

## Recommended Specs

- Format: PNG with transparent background or SVG
- Size: Square, 128×128px or 256×256px
- File size: Keep under 50KB
