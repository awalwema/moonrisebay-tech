# Project Demo Videos

Place short demo videos (30-60 seconds) here.

## Naming Convention

Use project name prefix:

- `hearlabs-demo.mp4`
- `sipandscroll-demo.mp4`
- `faithfilter-demo.mp4`

## Recommended Specs

- **Format:** MP4 (H.264 codec)
- **Duration:** 30-60 seconds
- **Resolution:** 1920×1080 (1080p) or 1280×720 (720p)
- **File size:** Keep under 10MB per video (compress if needed)
- **Frame rate:** 30fps or 60fps
- **Aspect ratio:** 16:9

## Compression Tips

If videos are too large, compress with:
- HandBrake (free)
- FFmpeg: `ffmpeg -i input.mp4 -c:v libx264 -crf 23 -preset medium output.mp4`
- Online tools: cloudconvert.com

## Tips

- Show the core feature/workflow
- Start with the app already open (no loading screens)
- Keep it focused on one key feature
- No audio needed (or keep it minimal)
- End with a clear result/outcome
