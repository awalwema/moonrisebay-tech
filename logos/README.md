# Company Logos

Place company logo images in this folder.

## Naming Convention

Use lowercase company names without spaces:

- `hearlabs.png` or `hearlabs.svg`
- `faithfilter.png` or `faithfilter.svg`
- `weavlog.png` or `weavlog.svg`
- `digitaleventsinc.png` or `digitaleventsinc.svg`

## Recommended Specs

- **Format:** PNG with transparent background or SVG
- **Size:** Square aspect ratio (e.g., 256×256px or 512×512px)
- **File size:** Keep under 100KB for fast loading

## Usage

Once you add the logo files, update `index.html` to reference them:

```html
<!-- Change from: -->
<img src="https://www.hearlabs.com/favicon.ico" ... />

<!-- To: -->
<img src="logos/hearlabs.png" ... />
```
